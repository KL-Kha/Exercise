DOWNTOWN MARKET

WEB&APP DESIGN

PROJECT OVERVIEW

Downtown Flea Market is a fictitious organization. Its main service is to host occasional events or flea markets where people can not only go shopping, but also relax, and enjoy a lot of exciting side activities.

The primary purpose of this project is to create an interactive prototyping system for DFW website, including 5 main pages: Homepage, About, Services/Ticket Pricing, Contact and FAQ page.
